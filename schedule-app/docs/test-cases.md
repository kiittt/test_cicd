## AI 建议后的待确认项

| 问题 | 当前状态 | 建议 |
| --- | --- | --- |
| 标题超长时是否限制长度 | 需求只写“标题超长”为异常场景，但未定义最大长度 | 先补充产品规则，例如 100 字符；再实现后端和前端校验 |
| 是否允许结束时间等于开始时间 | 当前规则只禁止开始时间晚于结束时间 | 保持允许，除非业务要求会议必须有持续时间 |
| 跨天日程按哪个日期筛选 | 当前实现按 `startTime` 所在日期筛选 | 文档中明确“日期筛选按开始日期” |
| 网络失败是否需要 UI 自动化覆盖 | 当前可通过组件测试模拟 fetch 失败，E2E 注入网络异常成本较高 | 后续优先补组件测试 |
| 服务端 500 是否需要强制模拟 | 当前正常路径无法稳定触发 500 | 后续通过测试专用 handler 或 mock 服务覆盖 |
